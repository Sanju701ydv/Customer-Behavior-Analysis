# Customer Behavior Data Analytics Project

## Overview

This project demonstrates an end-to-end **Data Analytics** workflow using **Python, SQL (PostgreSQL/MySQL/SQL Server), and Power BI**. It focuses on transforming raw customer shopping data into meaningful business insights through data preprocessing, exploratory data analysis (EDA), SQL querying, and interactive dashboard creation.

The project showcases the complete analytics pipeline, making it suitable for demonstrating practical data analytics skills.

---

## Dataset

The dataset contains customer shopping behavior information, including:

* Customer ID
* Age
* Gender
* Item Purchased
* Product Category
* Purchase Amount
* Location
* Size & Color
* Season
* Review Rating
* Subscription Status
* Shipping Type
* Discount Applied
* Promo Code Used
* Previous Purchases
* Payment Method
* Purchase Frequency

## Project Steps

### 1. Data Loading

* Imported the dataset into Python.
* Loaded data using Pandas.
* Explored dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Checked missing values and duplicates.
* Generated descriptive statistics.
* Analyzed customer demographics.
* Explored purchasing patterns.
* Visualized trends using charts and graphs.

### 3. Data Cleaning

* Removed duplicate records.
* Renamed inconsistent column names.
* Corrected data types.
* Handled missing or invalid values.
* Prepared the dataset for analysis.

### 4. SQL Analysis

The cleaned dataset was imported into PostgreSQL/MySQL/SQL Server to perform analytical queries, including:

* Total Revenue
* Total Customers
* Average Purchase Amount
* Revenue by Gender
* Revenue by Product Category
* Revenue by Location
* Average Review Ratings
* Subscription Status Analysis
* Payment Method Analysis
* Purchase Frequency Analysis

### 5. Power BI Dashboard

Created an interactive dashboard to visualize business insights using:

* KPI Cards
* Bar Charts
* Pie Charts
* Column Charts
* Maps
* Slicers and Filters
* Interactive Visualizations

## How to Run

1. Clone or download this repository.
2. Install the required Python libraries:

   * Pandas
   * NumPy
   * Matplotlib
   * Seaborn
   * SQLAlchemy
3. Open the Jupyter Notebook or VS Code project.
4. Run the notebook to perform EDA and data cleaning.
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL queries for analysis.
7. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
